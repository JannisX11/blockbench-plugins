class="animated-java-plugin-about-page">Animated Java is a plugin for Blockbench that allows you to create custom animations and models
	for Vanilla Minecraft. It uses the power of Java Edition's Data Pack and Resource Pack systems
	to bring your creations to life!</p> <h3 class="animated-java-plugin-page-h3 animated-java-plugin-about-page">Features</h3> <ul class="animated-java-plugin-about-page"><li class="animated-java-plugin-about-page">Variants: Swap between different textures in-game.</li> <li class="animated-java-plugin-about-page">Highly optimized: Hours of performance tests and optimization tweaks have gone into Animated
		Java's Exported Data Pack to make sure it has as little performance impact as possible.</li> <li class="animated-java-plugin-about-page">Limited Molang support. If Blockbench can render it, Animated Java can animate it.</li> <li class="animated-java-plugin-about-page">Text Display previewing and animation support.</li> <li class="animated-java-plugin-about-page">Resource Pack-less exporting. Animate Vanilla block and item models!</li> <li class="animated-java-plugin-about-page">Locators: Run commands relative to a locators position via keyframes.</li> <li class="animated-java-plugin-about-page">Camera Plugin Support: Install the <a href="https://www.blockbench.net/plugins/cameras" class="animated-java-plugin-about-page">Official Camera Plugin</a> to create cinematic camera paths with ease!</li> <li class="animated-java-plugin-about-page">Animation Tweening: Create smooth transitions between animations.</li> <li class="animated-java-plugin-about-page">Many different configuration options.</li> <li class="animated-java-plugin-about-page">Complete Documentation at <a href="https://animated-java.dev/docs" class="animated-java-plugin-about-page">animated-java.dev/documentation/</a></li> <li class="animated-java-plugin-about-page">And much more!</li></ul> <h3 class="animated-java-plugin-page-h3 animated-java-plugin-about-page">Getting started</h3> <p class="animated-java-plugin-about-page">To learn the basics of Animated Java, check out our <a href="https://animated-java.dev/docs/getting-started/installing-animated-java" class="animated-java-plugin-about-page">Getting Started</a> guide.</p> <p class="note animated-java-plugin-about-page"><i class="animated-java-plugin-about-page">Note that it is <b class="animated-java-plugin-about-page">highly</b> recommended to have a basic understanding of Java Edition's Data
