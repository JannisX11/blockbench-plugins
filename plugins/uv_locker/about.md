# About UV Locker

UV Locker automatically reprojects UV coordinates when you move vertices in 3D edit mode — just like Blender's **Fix Face Attributes** feature. No more distorted textures after tweaking geometry.

### How it Works

1. Enable **Lock UV** from the main toolbar or Settings.
2. Move, rotate, or resize vertices in 3D — UVs are reprojected automatically.
3. Optionally enable **Exclude Selected Faces** to leave explicitly selected faces untouched.

### Use Cases

- Adjusting proportions of a model without breaking texturing
- Fine-tuning block shapes in Minecraft models
- Any workflow where geometry and UVs need to stay in sync
