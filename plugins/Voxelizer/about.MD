# Voxelizer

⚠️ **IMPORTANT NOTICE / ATTENTION:**  
This initial version (1.0.0) is a **pre-release stub** created solely for repository registration and layout preparation in the Blockbench store. It does **NOT** contain functional conversion tools yet. Full features will be activated in the next upcoming release.

---

## Future Overview 
Once fully released, **Voxelizer** will be a powerful utility tool designed to convert complex **Mesh geometry** back into standard Minecraft **Java Cubes** and **Bedrock Models**.

## Planned Features
* **Universal Conversion:** Instantly turn any non-cube mesh face into a real, valid Java or Bedrock element.
* **Preserve Transformations:** Original positions, dimensions, and custom rotation angles will stay 100% intact.
* **Hierarchy Friendly:** Your existing Outliner folder organization and group structures will remain completely clean.
* **Smooth Workflow:** Full integration with Blockbench history for single-action Undo/Redo (`Ctrl + Z`).

## Roadmap
* **v1.0.0 (Current):** Repository structure, icon branding, and multi-platform registration (`both` variant for desktop & web).
* **v2.0.0 (Next):** Full activation of the functional Mesh-to-Cube conversion engine for Java and Bedrock projects.
