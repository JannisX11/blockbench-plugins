# About Fly Mode

Fly Mode adds FPS-style viewport navigation to Blockbench, letting you fly through your scene freely — just like in a first-person game.

### How it Works

1. Enable Fly Mode from the **View** menu or with the tool toggle.
2. Hold **Right Mouse Button** to activate flight control.
3. Move the mouse to **look around** (yaw and pitch).
4. Use **W/A/S/D** to move forward/left/backward/right.
5. Use **Space/Shift** to move up/down (rebindable).
6. Scroll the **mouse wheel** while flying to adjust movement speed on the fly.

### Configuration

- **Keybindings** — all movement keys are rebindable in the Keybindings menu
- **Speed** — base speed, faster/slower multipliers, and wheel range in Settings → Preview
- **Mouse sensitivity** — horizontal and vertical sensitivity, plus an invert-pitch toggle
