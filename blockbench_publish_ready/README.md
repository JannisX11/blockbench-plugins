# Blockbench Plugin Publication Package

This folder contains the public-ready files for the Dedou3D Blockbench plugins.

## Plugins

- `armor_exploded_view`
- `hitbox_guard`
- `name_guard`

## Contents

- `plugins/<plugin_id>/<plugin_id>.js`
- `plugins/<plugin_id>/about.md`
- `plugins.json` with entries ready to merge into the official Blockbench plugin list

## Next Steps

1. Fork the official Blockbench plugin repository.
2. Add each plugin folder under the repository `plugins` directory.
3. Merge the entries from this `plugins.json` into the repository `plugins.json`.
4. Test each plugin in Blockbench.
5. Open a Pull Request.
